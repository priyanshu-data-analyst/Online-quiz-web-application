# Online Ranking System (Quiz Web Application)

## Project Overview
The Online Ranking System is a web-based quiz platform that allows teachers to create quizzes and students to participate in them. The system automatically evaluates answers, calculates scores, and generates a real-time leaderboard to rank participants based on their performance.

The main goal of this project is to enhance student engagement through interactive quizzes and gamified learning methods.

---

## Features
- User registration and login system
- Teacher module to create and manage quizzes
- Student module to attempt quizzes
- Timer-based quiz attempts
- Automatic score calculation
- Real-time leaderboard and ranking system
- Performance tracking and analytics

---

## System Modules
### 1. User Management
Handles user authentication and role-based access (Teacher / Student).

### 2. Quiz Management
Teachers can create quizzes with questions, options, and correct answers.

### 3. Leaderboard System
Ranks students based on score and time taken.

### 4. Performance Analytics
Allows teachers to monitor student progress and quiz performance.

## How the System Works

1. User registers or logs into the system.
2. Teachers create quizzes with questions and time limits.
3. Students join quizzes using a unique quiz code.
4. Students answer questions within the given time.
5. The system calculates the score automatically.
6. The leaderboard updates with student rankings.
